# Consent-Based Emergency Signal Network

Snapshot is a local-first safety platform prototype for consent-based emergency signal snapshots, voluntary discovery, BLE/IoT safety device integration, and guardian-authorized emergency case workflows.

The project is designed around privacy, consent, anti-abuse controls, short retention defaults, and auditability. It does not support open-ended covert tracking, unauthorized identifier collection, or bypassing operating-system privacy restrictions.

Early MVP work focuses on a safe mobile app foundation with mock/local signal snapshots, BLE/network/location abstractions, snapshot history, emergency case placeholders, consent language, and QA-friendly debug/export views.

See `project.plan` for current scope, operating rules, agent roles, and build tasks.
